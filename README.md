# 🎵 Finish The Lyrics

Finish The Lyrics is an interactive music quiz game developed using Python and Pygame.

Players choose a music category and complete missing song lyrics while earning points and managing limited lives.

---

# 🎮 Features

- Multiple music categories:
  - Pop
  - Bollywood
  - Nursery Rhymes

- Interactive graphical user interface using Pygame
- Background music and sound effects
- Hint system
- Score tracking
- Lives system
- Performance rating screen
- Dynamic category backgrounds
- Replay functionality

---

# 🛠 Technologies Used

- Python
- Pygame

---


# ▶️ How to Run the Game

## Step 1 — Install Python

Download and install Python from:

https://www.python.org/downloads/

---

## Step 2 — Install Pygame

Open terminal or command prompt and run:

pip install pygame

---

## Step 3 — Run the Game

Open terminal in the project folder and run:

python3 main.py for terminal logic
python3 pygame_game.py for GUI

---

# 🎯 How to Play

1. Click START GAME
2. Choose a category
3. Complete the missing lyric
4. Press ENTER to submit your answer
5. Press H for a hint
6. Earn points for correct answers
7. Lose lives for wrong answers
8. The game ends when:
   - All questions are completed
   - All lives are lost
